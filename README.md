# TrustLens Live

TrustLens Live is a Chrome extension that provides real-time trust signals
for selected text on webpages, helping users detect misinformation without
leaving the page.

## Features
- Inline trust highlighting (Verified / Unverifiable / False)
- Floating trust result card
- Rule-based intelligent fallback (AI-ready)
- Chrome Extension (Manifest V3)
- Node.js + Express backend

## Tech Stack
- Chrome Extension (MV3)
- Node.js, Express
- DOM-safe content manipulation
- Explainable verification logic

## Demo Flow
1. Start backend
2. Load extension
3. Select text on any public webpage
4. View instant trust feedback

## Future Scope
- Gemini API integration
- Web search grounding
- Multilingual verification
